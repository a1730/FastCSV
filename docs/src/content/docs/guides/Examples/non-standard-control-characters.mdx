The CSV specification ([section 2.6. of RFC 4180](https://www.rfc-editor.org/rfc/rfc4180#section-2)) defines the following control characters:

- **Field separator**: `,`
- **Record delimiter**: `\r\n`
- **Field enclosure**: `"`

FastCSV uses these control characters by default. However, you can change them using the `CsvReaderBuilder` or
`CsvWriterBuilder` class.

while reading the CSV file.
:::
